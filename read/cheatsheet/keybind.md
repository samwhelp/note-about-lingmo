## 開啟應用程式 / Terminal

| 按鍵組合          | 功能         | 執行指令                     |
| ----------------- | ------------- | --------------------------- |
| `Alt + Enter`     | 開啟 Terminal | `lingmo-terminal`                 |
| `Alt + Shift + a` | 開啟 Terminal | `lingmo-terminal`                 |
| `Alt + Ctrl + a`  | 開啟 Terminal | `sakura`                 |
| `Alt + Shift + t` | 開啟 Terminal | `xfce4-terminal`                 |
| `Alt + Ctrl + t`  | 開啟 Terminal | `qterminal`                 |
